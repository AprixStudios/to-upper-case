> Makes a string fully uppercase.

### 🏠 [Homepage](https://github.com/AprixStudios/to-upper-case#readme)

## Install

```sh
npm install to-upper-case
```

## Example

```js
const upperCase = require('string-to-upper-case');

upperCase("oh no!"); // "OH NO!"

upperCase("Hey!"); // "HEY!"

upperCase("ily"); // "ILY"

upperCase("Yes."); // "YES."
```
